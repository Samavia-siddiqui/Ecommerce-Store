# 🛒 Ecommerce Store

A responsive ecommerce web application built with **React** and **Vite**.

## 🔗 Live Demo
👉 [View Live Site](https://ecommerce-store-chi-five.vercel.app/)

## 🚀 Features
- 🛍️ Product listing page
- 🛒 Add to cart functionality
- 📱 Fully responsive design
- ⚡ Fast performance with Vite

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite
- **Styling:** CSS
- **Deployment:** Vercel

## 📦 Run Locally

```bash
git clone https://github.com/Samavia-siddiqui/Ecommerce-Store.git
cd Ecommerce-Store
npm install
npm run dev
```
